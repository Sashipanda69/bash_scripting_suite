# Bash Scripting Suite for System Maintenance

Project Title: Bash Scripting Suite for System Maintenance

## Contents
- scripts/: Bash scripts for backup, update & cleanup, log monitoring, user/disk report, menu
- config/: configuration file
- reports/: generated reports
- screenshots/: sample screenshot images
- backups/: backup archives created by the script

## How to run
Edit config/config.cfg as necessary, then:
cd scripts
./menu.sh

Or run individual scripts:
./scripts/backup.sh
./scripts/update_cleanup.sh
./scripts/log_monitor.sh
./scripts/user_disk_report.sh

## Notes
This demo uses sandbox-friendly paths. Update config for production.
